# My Personal Website Portfolio

This is my personal website portfolio built using React and Vite.

The link to website - [link](https://my-portfolio432.vercel.app/)

![screenshot of personal-webp](Portfolio.jpg)

## Description

This is my personal website portfolio where I showcase my web development skills. It was built using React, and Vite for the frontend and Three.js to create the visual effects.

The website features a 3D animation header, a skills section, a timeline of my work experience, a projects section, and a contact form. The website is fully responsive and adapts to different device sizes.

Feel free to contribute to this project by forking this repository and submitting a pull request.

## Installation

1. Clone this repository
   ```
   git clone https://github.com/aroslavdmitriev/my-portfolio.git
   ```
2. Install the required dependencies using NPM
   ```
   npm install
   ```
3. Start the development server
   ```
   npm run dev
   ```

## Usage

To view the website locally, open your web browser and navigate to http://localhost:3000
